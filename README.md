# 🎌 Anime Data Visualization

A Python-based data analysis project that explores an anime dataset using **Pandas** and **Matplotlib**. This project focuses on transforming raw data into meaningful insights through multiple types of visualizations.

---

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an anime dataset, analyzing key attributes such as:

* Average ratings
* Genre distribution
* Anime types (TV, Movie, OVA, etc.)
* Popularity trends

The goal is to understand patterns and trends in anime data using visual storytelling.

---

## 📊 Visualizations Included

The project generates the following plots:

* 📊 **Bar Chart** – Average rating of each anime
* 📈 **Line Chart** – Rating trend across anime IDs
* 🥧 **Pie Chart** – Distribution of anime types
* 📉 **Histogram** – Distribution of ratings
* 🔺 **Stair Plot** – Step-wise rating visualization

---

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/anime-data-visualization.git
```

2. Navigate to the project folder:

```bash
cd anime-data-visualization
```

3. Install dependencies:

```bash
pip install pandas matplotlib
```

4. Run the script:

```bash
python animedatset.py
```
